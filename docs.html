API Documentation

          GET
          /health
        

        
Health check endpoint. Returns service status.

        
        

      


      
      

        

          GET
          /defi/cetes/rate
        

        
Live CETES rates from Etherfuse API. Returns APY, price in MXN, and XLM/USDC rate.

        
        

      


      
      

        

          GET
          /api/v1/cash/agents
        

        
List of P2P cash exchange agents. Requires x402 payment (0.001 USDC).

        
⚠️ Returns 402 Payment Required without x402 header

        
        

      


      
      

        

          POST
          /api/v1/cash/request
        

        
Request cash exchange. Body: { "amount": "100", "lat": 19.4, "lng": -99.1 }

        

          
Request Body:

          
{ "amount": "100", "lat": 19.4, "lng": -99.1 }

        

        
        

      


      
      

        

          GET
          /health/live
        

        
Kubernetes liveness probe endpoint.

        
        

      


      
      

        

          GET
          /health/ready
        

        
Kubernetes readiness probe. Checks DB, Stellar RPC, and contracts.
